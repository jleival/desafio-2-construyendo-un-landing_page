# 🌿 DecoPlantas
Proyecto web desarrollado con **HTML5 y CSS3**, enfocado en la creación de una **Landing Page comercial** para una tienda ficticia de plantas decorativas.
Este desafío fue realizado **sin Bootstrap ni frameworks externos**, utilizando únicamente tecnologías base del desarrollo web.

---

## 🚀 Demo del Proyecto

🔗 **GitHub Pages:**
*https://jleival.github.io/desafio-2-construyendo-un-landing_page/*


---
## 📌 Descripción
**DecoPlantas** presenta un catálogo visual de plantas decorativas para el hogar, mostrando productos con imagen, nombre y precio en una estructura limpia, ordenada y atractiva.

El proyecto fue creado para reforzar conocimientos en:

- Estructura HTML semántica  
- Maquetación básica con CSS  
- Organización visual de contenido  
- Buenas prácticas de desarrollo frontend  
- Uso de archivos externos de estilos  
---
